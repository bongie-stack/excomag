# Beacon of Life Institute Website

This is the official website for Beacon of Life Institute, featuring a landing page that directs users to either the Primary School or High School sections.

## Technologies
- **React** (via Vite)
- **React Router** (for navigation)
- **Vanilla CSS** (for styling)
- **Lucide React** (for icons)

## Prerequisites
- Node.js installed on your machine.

## Getting Started

1. **Install Dependencies**
   Open a terminal in this directory and run:
   ```bash
   npm install
   ```

2. **Run Development Server**
   Start the local development server:
   ```bash
   npm run dev
   ```

3. **Build for Production**
   To create a production-ready build:
   ```bash
   npm run build
   ```

## Project Structure
- `src/components`: Shared components like Header and Footer.
- `src/pages`: Individual page components (Landing, Primary, HighSchool).
- `src/index.css`: Global styles and strict theme variables.
